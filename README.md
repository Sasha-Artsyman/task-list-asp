Task list (ASP.NET)
==============
Form for creating, viewing, editing and deleting entries from the to-do list.

**Languages:** 
- `C#`
- `HTML`
- `CSS`
- `JS`

**Used:** 
- `ASP.NET`
- `Json`


==============
You can view the list of tasks, add tasks to it, delete tasks. You can also change the completion mark from the list form.
